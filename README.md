# Frontend — Sistema de Gestão de Locações

Frontend web do sistema de gestão de locação de produtos, desenvolvido em React, operando em ambiente on-premise.

## Objetivo
Interface web para autenticação, gestão de produtos, clientes, locações e contratos.

## Stack
- React
- React Router
- Axios
- UI Library (a definir)

## Autenticação
- JWT
- Controle de sessão
- Tratamento de 401/403

## RBAC
- Controle de rotas
- Exibição condicional de menus

## Telas
Login, Dashboard, Produtos, Clientes, Locações, Contratos, Usuários.

## Deploy
Build React servido em ambiente local.
